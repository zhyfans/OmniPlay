# 2026-05-06 首页扫描状态与占位海报修正

## 修正内容

- 本地、WebDAV、Plex、Emby、Jellyfin 等新增媒体源流程统一进入扫描/刮削状态，首页左上角会显示“正在扫描媒体源...”或对应预扫描信息。
- 扫描入库时只为能提取出可用影视名称的文件创建首页占位；纯发布参数、通用目录名、无法识别标题的新文件会跳过。
- 剧集文件优先使用文件名或有效父目录提取剧名，同一部剧的多集文件会归到一个剧集海报，不再按分集生成首页海报。
- 移除媒体源会立即取消当前扫描/刮削任务并隐藏首页任务状态，避免旧的刮削或分集剧照状态残留。
- 电影和剧集仓储读取首页数据时会过滤历史无效负 ID 占位，防止旧数据继续显示。
